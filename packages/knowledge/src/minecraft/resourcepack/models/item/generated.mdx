The `builtin/generated` model is a special, hardcoded model in Minecraft that takes a flat 2D texture and dynamically extrudes its opaque pixels into a 3D item.

## Usage

To use this model, set the `parent` property of your item model to `"item/generated"` (or `"builtin/generated"` in older versions) and define the textures.

```json
{
  "parent": "item/generated",
  "textures": {
    "layer0": "item/diamond_sword"
  }
}
```

## Properties
